## API

### 原型扩展

#### Array.prototype.distinct
- 类型：布尔值
- 默认：false
- 两种不同的实现，false速度更优

#### String.prototype.trim
- 类型：布尔值
- 默认：false
- true为去掉字符串内所有空格，false只去掉首位空格
